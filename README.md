#  Diamond Price AI

A machine learning web application that predicts diamond prices based on their characteristics.

##  Live Demo

Coming soon (Streamlit deployment)

---

##  Features

- Predict diamond prices using AI
- Interactive web interface
- Feature importance visualization
- Modern Streamlit dashboard

---

## Machine Learning Models

This project compares several regression models:

- K-Nearest Neighbors (KNN)
- Random Forest
- XGBoost

Evaluation metrics used:

- R² Score
- RMSE
- MAE
